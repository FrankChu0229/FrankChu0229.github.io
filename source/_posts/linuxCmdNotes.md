# 常用快捷键

- `Ctrl + a` 光标移动到行首
- `Ctrl + e` 光标移动到行尾
- `Ctrl + l` 清屏
- `Ctrl + Shift + c` 复制
- `Ctrl + Shift + v` 粘贴
- `Ctrl + r` 逆向搜索命令历史
- `Ctrl + c` 终止命令
- `Ctrl + z` 挂起命令
- `Ctrl + u` 删除光标左边所有元素
- `Ctrl + k` 删除光标右边所有元素
